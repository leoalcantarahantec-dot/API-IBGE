# API IBGE - Consulta de Nomes

Aplicação web desenvolvida com Streamlit que consome a API pública de nomes do IBGE.

## Funcionalidades

- Consulta frequência de nomes por década
- Consome API oficial do IBGE
- Interface simples com Streamlit

## Tecnologias utilizadas

- Python
- Requests
- Streamlit
- API pública do IBGE

## Como executar

1. Clone o repositório:
````bash
git clone https://github.com/leoalcantarahantec-dot/API-IBGE.git
cd API-IBGE
````

2. Instale as dependências:
````bash
pip install -r requirements.txt
````

3. Execute a aplicação:
````bash
streamlit run miniprojeto.py
````
